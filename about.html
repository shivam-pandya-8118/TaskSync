About the Project

Project NameTaskSync Project TypeWeb-based ToDo / Task Management Application TechnologiesHTML, CSS, JavaScript, Bootstrap 5, PHP, MySQL PurposeTo provide a clean, minimal and effective task management tool for daily productivity

Key Features:

  • Add, track and manage tasks with real-time status updates
  • Tri-state column sorting (Default → Ascending → Descending) with dynamic serial numbering
  • Color-coded status indicators - Gray (Not Started), Orange (Pending), Green (Completed)
  • Safe deletion with custom confirmation dialog and automatic serial number rearrangement
  • System-aware dark/light theme with persistent toggle
  • Fully responsive design with sticky table headers and scrollable task list
